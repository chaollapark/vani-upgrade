Just like the Verse Line search, this search is aimed at the verse content of five main books that Srila Prabhupada gave us: Bhagavad-gītā, Śrīmad-Bhāgavatam, Caitanya-caritāmṛta, Nectar of Instruction and Śrī Īśopaniṣad. To maximize the search speed, a database has been compiled, containing the verse lines, synonyms (English word by word translations) and a list of all individual words. So the searching is performed in the database and not in the Vanipedia pages, saving us from a lot of parsing complexity.

Momentarily the search is confined to the English versions. So the hyperlinks in the resulting verse references are leading to English pages. Since more and more translations are appearing, it is our aim to also provide the possibility to select a specific language in the future.

The "Whole Words" options can be used to limit the result to those cases where the search query only matches sequences of characters that are surrounded by so called word boundaries.

The "Diacritic Sensitive" option can be used to enforce an exact match with the words in the search query, regarding diacritic characters. The option only exists for the "Sanskrit or Bengali" part of the synonym, because the translation part only contains diacritic characters sporadically. If the option is not checked, a search term like karuna will match karuṇa and karuṇā and vice versa. If the option is checked, only exact matches are returned. Like the "Whole Words" option, this can be useful to limit the number of resulting verses.

A book filter is provided to limit the result to verses of a specific book.

The table with resulting verses can be sorted by any column: "Sanskrit or Bengali", "English" or "Verse Reference". This can be done by making a choice in the "Sort By" drop-down list, or by clicking a specific column header.

You can enter a search query and then either click the Search button, or press Enter, to execute the search.

Preceding the search-query textbox, is a drop-down list with several operators that determine how the search query is applied. In contrast to the Verse Line search, the list also contains some logical-negation operators (with "not"). This can be useful to limit the number of resulting verses. Because the Synonym search consist of two parts ("Sanskrit or Bengali" and "English"), the result is actually the intersection of two search queries.
